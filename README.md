# The Effect of Source Prestige on Perceived Credibility of Health Information
## Overview
This repository contains the published HTML version of my PSY 120L final reproducible
report.
This project examined whether source prestige influences perceptions of source credibility and whether this relationship is moderated by individual differences in Need for Cognition and Trust in Science and Scientists. Participants were randomly assigned to either a low- or high-prestige source condition, and their ratings of source competence, goodwill, and trustworthiness were compared across conditions and moderator groups..
## Report
The published report is available through GitHub Pages.
